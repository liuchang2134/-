# Price Action Academy

Al Brooks 价格行为学一站式学习网站。

## 内容

- 15 节主线课程学习路径
- 72 个 Brooks 价格行为形态百科条目
- 360 张 AB 原图中文标注样本
- 全视频字幕摘要库
- 全资料地图
- 形态练习题与训练清单
- 独立新版：136 个价格行为图表百科条目，544 张中文课件图，搭配 Brooks 字幕课综合讲解

## 本地运行

在项目上一级目录启动静态服务器：

```powershell
python -m http.server 8766 -d "E:\价格行为学"
```

然后打开：

```text
http://127.0.0.1:8766/al-brooks-pattern-dashboard/
```

新版图表百科入口：

```text
http://127.0.0.1:8766/al-brooks-pattern-dashboard/price-action-cn-encyclopedia/
```
